## Visual Standards

These baselines demonstrate:

1. **TailAdmin eCommerce dashboard pattern** - Metric cards, tables, cards
2. **Frozen TABLE_* constants** - Consistent table styling
3. **Proper spacing** - `space-y-6` between sections
4. **Card primitives** - All content in Card/CardHeader/CardBody
5. **Empty states** - CardEmptyState for zero-data scenarios

---

## Comparison Guidelines

When reviewing UI changes:

1. Compare new screenshots to these baselines
2. Verify spacing matches exactly
3. Check table header background (should be `bg-gray-100`)
4. Confirm button variants are consistent
5. Validate empty states exist for all lists/tables
